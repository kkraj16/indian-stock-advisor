# Indian-stock-advisor
StockAdvisor is a Java-based stock market analysis and advisory system focused on the Indian stock market (NSE/BSE). It fetches and stores historical and latest market data from third-party APIs, offers buy/sell indicators using technical and fundamental analysis, and tracks IPOs (upcoming and past 5 years).

✅ Key Features:
* -[x] Fetch & store historical stock data from Indian markets
* -[x] Update real-time prices and indicators for searched stocks
* -[ ] Technical indicators: RSI, MACD, EMA, Bollinger Bands
* -[ ] Buy/Sell signal suggestions
* -[ ] IPO Tracker (Upcoming & past 5 years)
* -[ ] User-friendly dashboard (web/mobile)
